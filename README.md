# playwright-ui-automation-practice
This repository contains end-to-end UI automation tests built using Playwright with JavaScript for the website:

https://testautomationpractice.blogspot.com/

The project demonstrates handling of various web elements and user interactions without using the Page Object Model (POM). It focuses on understanding Playwright fundamentals and UI automation concepts.
